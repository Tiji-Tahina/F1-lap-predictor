# graph.md — F1 Lap Predictor Architecture

## Module Dependency Graph

```
main.py
  ├── src/data.py
  │     └── src/config.py
  ├── src/features.py
  │     └── src/config.py
  ├── src/models.py
  │     └── src/config.py
  └── src/viz.py
        └── src/config.py
```

## Data Flow

```
[fastf1 API]
     │
     ▼
load_all_races() ──────────────────────────► DataFrame (raw laps, all races)
     │
     ▼
clean_laps() ──────────────────────────────► DataFrame (filtered, LapTime_s)
     │                                          • pit in/out removed
     │                                          • 60–200s window
     │                                          • key cols non-null
     ▼
build_features()
  ├── add_stint() ─────────────────────────► + "Stint" column
  │                                            (detects tyre-life reset)
  │
  ├── add_lap_frac() ──────────────────────► + "LapFrac" column
  │                                            (fuel-load proxy)
  │
  └── encode_features() ───────────────────► + Driver_enc, Compound_enc, GP_enc
                                               (LabelEncoder per categorical)
     │
     ▼
build_features() returns ──────────────────► X (DataFrame), y (Series), encoders (dict)
     │
     ▼
train_val_split() ─────────────────────────► X_train, X_val, y_train, y_val
     │                                          (Monaco + Monza = holdout)
     │
     ▼
build_model() + train() ───────────────────► fitted XGBRegressor
     │
     ▼
evaluate() ────────────────────────────────► preds, MAE, RMSE
     │
     ├──► plot_residuals() ────────────────► residuals.png
     │
     └──► plot_shap() ─────────────────────► shap.png
```

## Module Responsibilities

| Module | Responsibility |
|---|---|
| `src/config.py` | Single source of truth: races, hyperparams, feature names, paths, constants |
| `src/data.py` | Session loading via fastf1, cache setup, lap filtering and cleaning |
| `src/features.py` | Stint detection, lap-fraction calculation, label encoding, X/y assembly |
| `src/models.py` | Race-based train/val split, XGBRegressor construction, training loop, MAE/RMSE |
| `src/viz.py` | Predicted-vs-actual scatter, SHAP summary importance plot |
| `main.py` | Wires modules together in pipeline order |

## Output Files

| File | Generated By |
|---|---|
| `residuals.png` | `src/viz.py::plot_residuals()` |
| `shap.png` | `src/viz.py::plot_shap()` |
| `f1_cache/` | fastf1 (auto-created, gitignored) |
